# discreet_math_20261_project
The project statement of the discreet math class of javeriana university 2026-1.

## 
The code in this repository is merely to help in implementing. Don't feel like you should use
	it. However; it may be important to implement the code in a way that it can be called as
	a class "PoliticalSAT" and that it has the methods __init__ ,solve_stability,generate_dnf.

The 'run_test.cmd' runs the cases in '/cases' and compares if the feasibility was found
	correctly. The feasibility is compared with the files '/cases/ou_0i.txt', which where
	generated by my code when given its corresponding '/cases/in_0i.txt' as input.
Use it as you may.

The 'main.py' is basically the interface to the solver. It has a way to verify if, given
	a decision it correctly solves the problem with all the requisites.
